# Reden 开源网站对比分析及修改计划

## 参考网站特点分析 (reden-website)

这是一个功能非常完善的 Minecraft 相关网站（Vue/Nuxt + Vuetify），包含以下关键功能：

### 核心功能对比

| 功能               | 参考网站                 | 当前项目 | 优先级   |
| ---------------- | -------------------- | ---- | ----- |
| **3D 蓝图预览**      | 使用 deepslate 库在浏览器渲染 | 无    | **高** |
| **点赞/踩功能**       | ✅                    | 无    | **中** |
| **下载统计**         | 显示总下载次数              | 无    | **高** |
| **卡片悬停效果**       | 悬停显示描述信息             | 无    | **中** |
| **多语言支持**        | 完整 i18n（en, zh, ru）  | 无    | **低** |
| **用户中心**         | 个人蓝图管理               | 无    | **中** |
| **标签系统**         | 分类标签 + 特征标签          | 已有   | -     |
| **Markdown 编辑器** | 用于描述信息               | 无    | **低** |
| **广告系统**         | 侧边栏广告位               | 无    | **低** |
| **创作者收益**        | 打赏/收益系统              | 无    | **低** |

***

## 修改计划

### 任务 1：添加下载统计功能（高优先级）

**文件**：`js/gallery.js`, `css/style.css`

**修改内容**：

* 在 `data.json` 添加 `downloads` 字段

* 卡片底部添加下载次数显示

* 模态框添加下载次数详情

```javascript
// 卡片底部添加
<span class="downloads">
  <v-icon size="18">mdi-download</v-icon>
  {number2text(bp.downloads)}
</span>
```

### 任务 2：添加点赞功能（中优先级）

**文件**：`js/gallery.js`, `worker-upload.js`

**修改内容**：

* localStorage 存储点赞状态

* 卡片添加点赞按钮

* 上传时存储点赞数

### 任务 3：卡片悬停效果（中优先级）

**文件**：`css/style.css`

**修改内容**：

* 悬停时显示蓝图描述

* 添加平滑过渡动画

* 悬停放大效果

### 任务 4：3D 蓝图预览（高优先级 - 可选）

**文件**：`converter.html`, `js/preview.js`

**方案**：集成 deepslate 库

**注意**：这是一个较大的工程，需要加载 Minecraft 材质和模型资源。对于 GitHub Pages 静态站，可能需要使用 CDN 加载资源。

### 任务 5：用户个人中心（中优先级）

**文件**：新增 `profile.html`

**功能**：

* 显示用户上传的蓝图

* 管理收藏的蓝图

* 编辑个人信息

***

## 实施步骤

1. **下载统计功能**

   * 修改 `data.json` 添加 `downloads` 字段

   * 修改 `gallery.js` 渲染下载次数

   * 添加 CSS 样式

2. **点赞功能**

   * 修改 `gallery.js` 添加点赞按钮和逻辑

   * 使用 localStorage 存储点赞状态

3. **卡片悬停效果**

   * 修改 `style.css` 添加悬停样式

   * 添加描述覆盖层

4. **3D 预览功能（可选）**

   * 调研 deepslate CDN 资源

   * 添加预览组件

5. **用户个人中心（可选）**

   * 创建 `profile.html`

   * 实现用户蓝图管理

***

## 决策确认

| 功能        | 选择             | 说明                                    |
| --------- | -------------- | ------------------------------------- |
| **3D 预览** | ✅ deepslate 库  | 集成 deepslate 库实现 Litematic 文件 3D 渲染预览 |
| **下载统计**  | ✅ localStorage | 本地存储下载次数，简单实现                         |

***

## 假设与限制

* 当前项目是纯前端静态网站（GitHub Pages）

* 部分功能（如下载统计服务器端存储）需要后端支持

* 3D 预览需要加载 Minecraft 材质资源，可能导致加载慢

***
